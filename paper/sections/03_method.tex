\section{Corpus and Methodology}
\label{sec:method}

\subsection{Reproducible environment}

The committed corpus records compiler, version, architecture, and CT-KAT commit
per build cell in \texttt{docs/corpus/corpus\_cells.csv}. Since a verdict can
depend on compiler and optimization level, prose without this provenance is
under-specified. The artifact workflow checks the regression suite; timing tests
may skip outside native, non-emulated setups.

\subsection{Corpus design}

The corpus pairs each layer with a case \emph{only} that layer flags, grounding
every verdict class: real PQClean targets~\cite{pqclean} plus synthetic
controls (Table~\ref{tab:coverage}, per-feature map;
Table~\ref{tab:corpus}, verdict-class corpus).

\paragraph{Real PQC targets.}
For ML-KEM-768~\cite{fips203} we screen the plain PQClean build and a
KyberSlash~\cite{kyberslash} division variant. The plain build passes
structurally; its FIPS~202/Keccak division candidates triage public. The variant
passes Memcheck but exposes a secret-derived division to \texttt{asm-scan}.

\paragraph{Synthetic controls.}
Three harnesses isolate one phenomenon each: \texttt{toy\_lookup} for a
secret-indexed lookup, \texttt{ct\_matrix\_flip} for an optimization-dependent
branch, and \texttt{toy\_dudect} for a timing-only control (Table~\ref{tab:dudect}).

\subsection{How a row is produced}

Each row follows one pipeline (Fig.~\ref{fig:pipeline}) from one YAML config:
\texttt{ct-matrix} reruns structural checking per build cell, \texttt{asm-scan}
records warn-only variable-latency candidates, triage applies default-deny, and
optional \texttt{dudect}~\cite{dudect} runs on timing distributions. The central
honest result is ML-DSA-65: structural checking fails, but default-deny holds it
at \texttt{needs-analysis} rather than declaring a leak.

\subsection{Merging the evidence}

Structural and \texttt{asm-scan} signals are per build cell, while timing is per
target. The merge script collapses them into Table~\ref{tab:corpus}; the timing
appendix remains Table~\ref{tab:dudect}.
